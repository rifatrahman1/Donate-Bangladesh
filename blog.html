Important FAQ

What is the Document Object Model (DOM?)

The Document Object Model (DOM) is a structured representation of an HTML or XML
document loaded in a browser. It represents every part of the document (like tags, attributes,
and text content) as nodes in a tree-like structure.Developers can use the DOM to
programmatically access, modify, or manipulate a document's content, structure, and styles.

How do you select an element from the DOM ?
To select an element from the DOM, you can identify it using its ID, class, tag name, or CSS
selector. For example, if you have a paragraph in your HTML document, you can select it using JavaScript
methods like getElementById, getElementsByClassName, querySelector, etc.
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where instead of attaching event listeners to multiple
child elements, you attach a single event listener to a common parent element. The parent element uses the
event bubbling mechanism to handle events triggered by its child elements.

How do you manipulate an eliment's attributes and style using the DOM?

To manipulate an element's attributes and styles using the DOM, JavaScript provides various methods and
properties. For example, you can change a paragraph's class, id, or its inline style to modify its appearance or behavior dynamically.
